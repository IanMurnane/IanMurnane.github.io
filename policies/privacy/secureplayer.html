Privacy Policy - Secure Media Player and File Manager

Last Updated: July 21, 2026

Welcome to Secure Media Player and File Manager. Your privacy is important to us. This Privacy Policy explains how the application operates, what information and device permissions are processed while using the app, and how integrated third-party services are used.

1. App Purpose and Privacy Approach

Secure Media Player and File Manager is designed to act as a core local file manager, coupled with privacy-conscious image, video, and audio playback.

The app focuses on providing deep local file tree navigation and media functionality. It aims to minimise unnecessary data collection and online activity while maintaining reliability and supporting ongoing development.

2. Permissions and Information Processed by the App

All Files Access (MANAGE_EXTERNAL_STORAGE)

To fulfill its primary purpose as a file manager, this app requires the All Files Access (MANAGE_EXTERNAL_STORAGE) permission.

This permission is strictly required to provide the core File Explorer interface. It allows users to browse, list, rename, delete, and organise media files across complex directory structures on both internal and external storage. Because standard system file pickers do not allow for continuous, multi-directory file management or the playback of corrupted media streams, this broad access is essential for the application to function.

Media Files and Local Processing

All media files and directory data accessed via the file manager interface are processed strictly locally on your device.

The developer does not collect, upload, transmit, analyse, or store the contents of your videos, images, audio files, playback history, or file structures on external servers. The storage access permission is never used for undisclosed purposes, analytics, or advertising.

Operational Diagnostics and Reliability Information

The app may collect limited operational and diagnostic information to help maintain reliability, improve performance, and identify technical issues.

Examples may include:

  • App launches and startup events
  • Error reports and diagnostic information
  • Technical logs and stack traces generated during failures
  • Device model and operating system version
  • Performance and stability indicators

This information is used solely to understand application behaviour, diagnose faults, and improve app quality. It is never used to analyse or collect media content or local file data.

3. Advertising and Third-Party Services

The app displays advertisements to support ongoing development. Advertisements may be provided through third-party advertising services such as Google AdMob.

Advertising services may process information in accordance with their own privacy practices and applicable laws. This may include:

  • IP address
  • Device information
  • Operating system version
  • Advertising identifiers where available
  • Consent preferences

Where required by law, users may be presented with a consent choice before personalised advertising is enabled.

Use of advertising-related information is governed by the applicable third-party provider privacy policies. No local file data or media content is ever shared with advertising networks.

4. Media Processing Components

The application includes embedded open-source media processing libraries, including FFmpeg, to provide audio and video functionality. These components operate strictly locally on your device and do not transmit any file data or media content to the developer or third parties.

5. Children’s Privacy

This application is not directed toward children under the age of 13. We do not knowingly collect personal information from children. If you believe information has been provided unintentionally, please contact us.

6. Data Security and Retention

Diagnostic information may be retained for a limited period to support reliability, maintenance, and issue resolution. We take reasonable measures to minimise unnecessary data collection and protect application integrity. Local file data is retained only on your device until manually deleted by the user via the file manager interface.

7. Changes to this Privacy Policy

This Privacy Policy may be updated periodically to reflect operational, legal, or product changes. The latest version will always be published with an updated revision date.
